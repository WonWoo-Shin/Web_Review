사용법

헤드태그안에 스타일태그를 넣어 사용

선택자

  • 전체 선택자 : 문서 전체에 스타일을 적용한다. 단 문서 안의 모든 요소를 읽어내기 때문에 잘 사용하지 않음
  • 태그 선택자 : 스타일을 적용할 태그를 선택한다.
  • 클래스/ID 선택자 : html문서 내에 클래스/ID가 정해진 태그만 스타일을 적용

속성

  • color : 글자의 색상을 적용
  • border : 개체에 박스모델을 적용
  • padding : 개체 내부의 폭을 선택
  • margin : 개체 외부의 폭을 선택
  • display : 화면에 표시될 방법을 선택
