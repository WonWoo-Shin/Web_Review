html 태그

HTML언어를 사용하여 작성한 코드 구역을 정의하는 태그

head 태그

  • title : 브라우저에 표시되는 이름을 정의하는 태그
  • method : 인코딩 방식을 정의하는 태그
  • stlye : CSS언어를 사용하여 HTML문서에 스타일을 추가하는 태그

body 태그

  • p : 문단을 구분하는 구역을 정의하는 태그
  • br : 줄 바꿈을 사용하는 태그
  • h1 ~ h6 : 제목을 구성하는 태그
  • strong : 글자를 진하게 설정하는 태그
  • u : 밑줄을 설정하는 태그
  • img : 사진을 삽입하는 태그
